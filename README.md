# DigitalStore

Интернет-магазин цифровых товаров на Django 5.2: каталог с фильтрами, корзина на сессиях,
заказы, оплата (демо-режим или ЮKassa) и ограниченные по сроку ссылки на скачивание.

## Требования

- Python 3.12+
- Зависимости из `requirements.txt`

## Запуск локально

1. Создайте виртуальное окружение:
   ```
   python -m venv .venv
   source .venv/bin/activate   # Linux/Mac
   .venv\Scripts\activate      # Windows
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` на основе `.env.example` и заполните его.

4. Выполните миграции:
   ```
   python manage.py migrate
   ```

5. Создайте суперпользователя:
   ```
   python manage.py createsuperuser
   ```

6. Запустите сервер:
   ```
   python manage.py runserver
   ```

7. Откройте http://127.0.0.1:8000. Товары и категории добавляются через
   админку: http://127.0.0.1:8000/admin/

## Деплой (Render или любой сервер с gunicorn)

Переменные окружения на сервере:

| Переменная | Значение |
|---|---|
| `SECRET_KEY` | случайный секретный ключ |
| `DEBUG` | `False` |
| `ALLOWED_HOSTS` | домены через запятую (на Render подхватывается автоматически) |
| `YOOKASSA_SHOP_ID`, `YOOKASSA_SECRET_KEY` | ключи ЮKassa (необязательно) |

Команды:

- Build: `./build.sh` (устанавливает зависимости, собирает статику, применяет миграции)
- Start: `gunicorn config.wsgi:application`

Статику в продакшене раздаёт WhiteNoise, отдельный веб-сервер не нужен.

## Оплата

Без ключей ЮKassa оплата работает в демо-режиме: заказ подтверждается кнопкой
на странице оплаты. Если ключи заданы, покупатель перенаправляется на страницу
ЮKassa, а статус заказа подтверждается через webhook (`/payments/webhook/`)
или прямой проверкой платежа при возврате в магазин. Для работы с ЮKassa
установите пакет: `pip install yookassa`.

## Структура проекта

- `config/` — настройки Django
- `products/` — товары и категории
- `orders/` — заказы и скачивание файлов
- `cart/` — корзина (сессии)
- `users/` — пользователи и профили
- `payments/` — оплата
- `templates/` — HTML-шаблоны
- `static/` — CSS и JS
